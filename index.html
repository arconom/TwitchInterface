Word Commands

pracro : Generate an acronym, pass the word to acronymize
prcacro : Generate an acronym using common words, pass the word to acronymize
prallit : Generate an alliteration, pass the first letter and then the number of words
prcallit : Generate an alliteration using common words, pass the first letter and then the number of words
prcword : Generate a random common word, or up to 10 if a number is given
prcwords : Generate a random common word, or up to 10 if a number is given
prcwordle : start a Wordle using a common word
prdef : Generate a random word and definition, or up to 10 if a number is given
prdefs : Generate a random word and definition, or up to 10 if a number is given
prdefine : Get the definition of the given word, if no word is given, get the definition of the most recently generated random word
prguess : guess a Wordle, prguess word
prword : Generate a random word, or up to 10 if a number is given
prwords : Generate a random word, or up to 10 if a number is given
prwordle : start a Wordle using any english word
prwordlestatus : show the current status of the Wordle
